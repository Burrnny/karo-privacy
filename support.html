Preguntas frecuentes

¿Cómo restauro mi compra de Klaro Pro?

Ajustes (engrane arriba a la derecha) → Klaro Pro → "Restaurar compras". Asegúrate de estar firmado con el mismo Apple ID con el que compraste.

Reportar una pregunta incorrecta

Toca el ícono de bandera ⚐ arriba a la derecha durante el quiz. Selecciona el motivo (la respuesta es otra, no se entiende, ortografía, opciones ambiguas, otro). Tu reporte es anónimo.

Activar o desactivar las notificaciones

Ajustes → "Recordatorio diario". También puedes ajustar la hora desde ahí.

Borrar mi progreso

Ajustes → "Borrar progreso" (botón rojo al final). Esta acción es permanente.

Mi racha se rompió

Si tenías un protector ❄️ disponible, se aplicó automáticamente. Recibes uno gratis cada mes.
